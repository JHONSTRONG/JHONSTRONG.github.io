windows curl安装及常规命令使用

发表于 2019-02-16 | 分类于 软件安装 | 评论数：

curl是什么

curl是一个利用URL语法在命令行下工作的文件传输工具，它支持的通信协议有FTP、FTPS、HTTP、HTTPS、TFTP、SFTP、Gopher、SCP、Telnet、DICT、FILE、LDAP、LDAPS、IMAP、POP3、SMTP和RTSP。curl还支持SSL认证、HTTP POST、HTTP PUT、FTP上传, HTTP form based upload、proxies、HTTP/2、cookies、用户名+密码认证(Basic, Plain, Digest, CRAM-MD5, NTLM, Negotiate and Kerberos)、file transfer resume、proxy tunneling

安装步骤

下载Windows版本的curl

  1. 下载Win版curl的地址如下https://curl.haxx.se/windows/
  2. 根据你自己的个人系统位数，下载对应的版本，我是X64的因此下载了下图中的红圈标注出来的版本
  3. 下载完成后解压，放到你想放置的位置，本次安装放在了D:\Program Files\curl（这里进行了解压文件的重命名，也可以不用重命名）

添加环境变量

  1. 我的电脑->右键“属性”->高级系统设置->环境变量
    在系统变量中点击新建按钮，如下图所示
  2. 弹出新建系统变量界面，在变量名中输入CURL_HOME，在变量值中输入D:\Program Files\curl\bin这个地方需要注意，替换成你自己的curl安装目录，并在最后添加\bin，具体如下图所示
  3. 修改系统变量中的Path，添加%CURL_HOME%，具体操作见下图
  4. 添加完成后，点击确定

    测试是否安装成功

  5. 打开CMD窗口，然后输入curl --help，看是否有反应，如果有如下输出，说明正常，如果没有，说明安装失败，根据具体的提示百度确认问题出在哪里

常用命令

详细的使用方法见地址https://curl.haxx.se/docs/manpage.html
语法：#curl [option] [url]

使用案例

  1. 获取页面的数据
    curl www.baidu.com

  2. 获取页面数据的同时，查看HTTP头信息
    curl -i www.baidu.com

  3. 仅获取HTTP头信息
    curl -I www.baidu.com

  4. 获取更详细的信息，包解析ip的过程等等
    curl -v www.baidu.com

  5. 保存访问的网页(文件会下载到你当前命令执行的所在目录)

  • 使用重定向功能保存
    curl www.baidu.com >> index.html

  • 使用curl内置option:-o(小写) 保存网页
    curl -o temp.html www.baidu.com

  • 使用curl内置option:-O(大写) 保存网页上的文件
    curl -O http://www.linux.com/hello.sh

  1. 测试网页返回值
    测试网站是否正常的常用方法
    curl -o /dev/null -s -w %{http_code} www.linux.com

  2. 通过执行代理服务器及端口访问
    curl -x 192.168.100.100:1080 http://www.linux.com

  3. Cookie
    很多网站上使用cookie记录session信息，curl可以增加相关参数处理cookie信息

  • 保存网站上的cookie，内置option: -c(小写)，文件会存储在你命令执行的位置
    curl -c cookie.txt www.baidu.com

  • 保存http的response里面的header信息，内置option:-D
    curl -D cookie.txt www.baidu.com

  • 使用cookie
    网站访问有时候会检测你的cookie信息是否合法，因此需要使用上面保存的cookie信息，内置option:-b
    curl -b cookie.txt www.baidu.com

  1. 模拟浏览器
    有些网站需要特定的浏览器进行访问，而且还有些需要特定的版本。curl内置option:-A可以让我们指定浏览器去访问网址

  2. 伪造referer（盗链）
    很多服务器会出现检测http访问的rederer从而控制访问。比如：先访问主页，然后再访问主页上的邮箱页面，这里访问邮箱的referer地址就是访问首页成功后的页面地址，如果服务器发现对邮箱页面访问的referer地址不是首页地址，断定是盗链
    curl内置option:-e 可以让我们设定referer

  3. 下载文件

  • 利用curl下载文件
    使用内置option: -o (小写)
    curl -o baidulogo.png https://www.baidu.com/img/superlogo_c4d7df0a003d3db9b65e9ef0fe6da1ec.png

  • 使用内置option: -O (大写)，这个会保存服务器上文件的名字作为本地文件名

IDEA中配置的方法

  1. 点击下图中的Edit Configurations，弹出Run/Debug Configurations对话框
  2. 选择你的Tomcat中间件，点击中间件上的Deployment，并点击绿色的加号
  3. 选择External Source，打开选择路径界面，选择一个路径进行部署
  4. 点击Apply按钮，然后点击Ok就可以了
